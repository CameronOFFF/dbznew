# Descrição do layout

- **Topo:** avisos da temporada com countdown de evento.
- **Coluna esquerda:** imagem do mundo + card do jogador com HP/KI/Stamina, atributos e moedas.
- **Coluna central:** cards de atividades (Arena, Missões, Treino, Recompensa diária) e tabs de conteúdo.
- **Coluna direita:** menu vertical com ícones + widget de ranking top 5.
- **Busca global:** barra com ícone de lupa para filtrar missões, itens e jogadores.

Estilo visual: tema escuro, bordas suaves, realces dourados (`glow`) e azuis (`nebula`).
